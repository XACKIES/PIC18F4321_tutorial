## Setup PIC18F4321

![image_1](https://github.com/XACKIES/PIC18F4321_tutorial/blob/main/Doc/IMG_1184.jpg)


- LED line     :  RD0-RD7 ,and RC4-RC5
- Pink LED     :  RC1
- Switch       :  RC3RB0-RB1
- Analog input :  AN8
- UART         :  RC7(RX) , RC6(TX)
